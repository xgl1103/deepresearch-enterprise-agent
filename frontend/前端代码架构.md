# 前端代码架构

## 项目概述
基于 React + TypeScript + Vite 构建的现代化前端应用，使用 TailwindCSS 进行样式设计。

## 核心技术栈
- React 19
- TypeScript
- Vite 构建工具
- TailwindCSS 样式框架
- Radix UI 组件库
- Lucide React 图标库
- React Router 路由管理
- @langchain/langgraph-sdk 用于与后端通信

## 文件结构与用途详解

### 根目录文件
- [index.html](./index.html) - 应用入口 HTML 文件，定义了页面标题和根节点
- [package.json](./package.json) - 项目依赖和脚本配置
- [vite.config.ts](./vite.config.ts) - Vite 构建配置文件
- [tsconfig.json](./tsconfig.json) - TypeScript 配置文件

### src 目录
#### 主应用文件
- [main.tsx](./src/main.tsx) - React 应用入口文件，负责渲染根组件
- [App.tsx](./src/App.tsx) - 主应用组件，包含核心业务逻辑
- [global.css](./src/global.css) - 全局样式文件，包含 Tailwind 配置和自定义样式

#### 组件目录 ([components/](./src/components/))
- [WelcomeScreen.tsx](./src/components/WelcomeScreen.tsx) - 欢迎界面组件，显示初始状态下的输入表单
- [ChatMessagesView.tsx](./src/components/ChatMessagesView.tsx) - 聊天消息视图组件，显示用户和 AI 之间的对话
- [InputForm.tsx](./src/components/InputForm.tsx) - 输入表单组件，包含文本输入、努力程度选择和模型选择
- [ActivityTimeline.tsx](./src/components/ActivityTimeline.tsx) - 活动时间线组件，显示研究过程中的各个步骤

#### UI 组件目录 ([components/ui/](./src/components/ui/))
这是一组基于 Radix UI 构建的可复用 UI 组件：
- [button.tsx](./src/components/ui/button.tsx) - 按钮组件
- [textarea.tsx](./src/components/ui/textarea.tsx) - 文本域组件
- [select.tsx](./src/components/ui/select.tsx) - 下拉选择组件
- [scroll-area.tsx](./src/components/ui/scroll-area.tsx) - 滚动区域组件
- [card.tsx](./src/components/ui/card.tsx) - 卡片组件
- [badge.tsx](./src/components/ui/badge.tsx) - 徽章组件
- [input.tsx](./src/components/ui/input.tsx) - 输入框组件
- [tabs.tsx](./src/components/ui/tabs.tsx) - 标签页组件

#### 工具库目录 ([lib/](./src/lib/))
- [utils.ts](./src/lib/utils.ts) - 工具函数，主要包含样式合并功能

## 核心功能流程

1. **应用初始化**：
   - [main.tsx](./src/main.tsx) 渲染 [App.tsx](./src/App.tsx) 组件
   - [App.tsx](./src/App.tsx) 使用 `useStream` hook 连接后端服务

2. **用户交互**：
   - 用户在 [InputForm.tsx](./src/components/InputForm.tsx) 中输入问题并选择参数（努力程度、模型）
   - 表单提交后通过 `thread.submit()` 发送请求到后端

3. **后端通信**：
   - 使用 [@langchain/langgraph-sdk] 与后端通信
   - 开发环境连接 `http://localhost:2024`，生产环境连接 `http://localhost:8123`

4. **结果显示**：
   - 通过 `onUpdateEvent` 回调函数接收研究过程中的各个步骤信息
   - 通过 `messages` 状态接收最终的 AI 回复
   - 使用 [ChatMessagesView.tsx](./src/components/ChatMessagesView.tsx) 展示对话内容
   - 使用 [ActivityTimeline.tsx](./src/components/ActivityTimeline.tsx) 展示研究过程

## 样式与主题
- 使用 TailwindCSS 进行样式设计
- 实现了深色主题适配
- 定义了丰富的动画效果和过渡效果

## 构建与开发
- 使用 Vite 进行快速开发和构建
- 配置了开发环境代理，将 `/api` 请求代理到 `http://127.0.0.1:8000`
- 支持 TypeScript 类型检查

这个前端应用结构清晰，组件化程度高，充分利用了现代前端技术栈的优势，为用户提供了一个流畅的 AI 研究助手交互体验。
